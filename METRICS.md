# TokenGuard Metrics

This file defines the product metrics that matter for TokenGuard.

---

## North Star

- Qualified audits per week: completed audits with at least one actionable recommendation and either a report share or a lead capture.

## Product Metrics

### Primary

- Audit completion rate
- Recommendation conversion rate
- High-savings lead capture rate
- Public report share rate
- Share-to-lead conversion rate

### Secondary

- Average monthly spend audited
- Average estimated annual savings
- Number of tools per audit
- Percentage of low-spend audits

## Operational Metrics

- Type-check pass rate
- Test pass rate
- Build pass rate
- Error rate in optional integrations

## Report Metrics

- Current monthly spend
- Current annual spend
- Estimated monthly savings
- Estimated annual savings
- Savings percentage
- Number of opportunities found

## Decision Metrics

- Percentage of audits that surface $250+/month in savings
- Percentage of audits that are already optimized
- Median savings per audited team
- Percentage of reports that are opened more than once

## Success Targets

- Clear spend and savings reporting on every audit
- No mismatch between savings labels and spend labels
- Fast, reliable report generation
- Meaningful lead capture on large savings opportunities
